## Band Name Generator

A small web application that generates fun band names based on user input.  
Built to practice Node.js, Express, and EJS templating with basic form handling.

### Tech Stack

- Node.js and Express.js for the backend server and routing  
- EJS templates with HTML and CSS for rendering dynamic pages  
- Body-Parser middleware for handling form data from the client

### Features

- Accepts user input (e.g., words or phrases) through a simple HTML form  
- Generates and displays a band name using server-side logic and EJS views  
- Clean, minimal UI suitable for demonstrating full-stack JavaScript basics
